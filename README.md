## Technical Skills
* Language: JSON (Behavior and Resource Packs)
* Scripting: JavaScript / JSON format (Minecraft Scripting API)
* Tools: Blockbench (Modeling/Animation), Bridge v2, VS Code

## Common Installation Instructions
1. Navigate to the specific project folder or the Releases tab.
2. Download the required .mcaddon or .mcpack files.
3. Import the files into Minecraft.
4. Enable the necessary Experimental Toggles in your world settings (refer to individual project documentation for specifics).
